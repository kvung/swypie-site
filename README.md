# swypie-site

Public legal & support pages for **Swypie: Photo Cleaner** (iOS). Static HTML served via GitHub Pages.

- Privacy Policy — https://kvung.github.io/swypie-site/privacy.html
- Support & FAQ — https://kvung.github.io/swypie-site/support.html

Plain HTML, no build step. Edit the `.html` files and push to `main`; GitHub Pages redeploys automatically.
